# Employee Management System (EMS)

A comprehensive web application for managing employee holidays, leave requests, and overtime tracking.

## Features

### For Employees
- Dashboard with statistics
- Leave request management
- Overtime tracking
- Holiday calendar
- User profile settings

### For Managers/Admins
- Request approvals
- Employee management
- Holiday management
- Analytics dashboard

## Tech Stack

- HTML5, CSS3, JavaScript
- PHP 7.4+
- MySQL 5.7+

## Installation

1. Import `database.sql` into MySQL
2. Update database credentials in `config.php`
3. Place files in web server directory
4. Access via http://localhost/ems/

## Demo Credentials

- Admin: admin@example.com / admin123

## File Structure

- config.php - Database configuration
- login.php - Login page
- index.php - Dashboard
- leave.php - Leave management
- overtime.php - Overtime tracking
- holidays.php - Holiday calendar
- approvals.php - Manager approvals
- profile.php - User settings
- sidebar.php - Navigation
- topbar.php - Top bar
- logout.php - Logout
- styles.css - Styling
- database.sql - Database schema
